# AI FinOps: Anthropic Claude Cost Framework

A guide for understanding, diagnosing,  and optimizing Anthropic Claude costs across seat subscriptions, 
API usage, and Claude Code.

**[Open the app →](https://gauravkantgoel.github.io/ai-finops-claude/)**

## What's inside

- **Learn**: Three spending streams, token economics, plans & pricing, 
  API anatomy, Claude Code costs, glossary
- **Diagnose**: Plan selector, console walkthrough, benchmarks, 
  common mistakes
- **Optimize**: Caching/routing/batch levers, before/after calculator, 
  FinOps maturity model

## Design principles

- Every pricing claim is badged as Official, Opinionated, or Estimate
- "Last verified" timestamps on all pricing blocks
- No external dependencies beyond Google Fonts
- Works in restrictive IT environments (single HTML file, no build step)
- Print-friendly (File → Print → Save as PDF)

Pricing data verified against claude.com/pricing and 
platform.claude.com/docs on April 10, 2026.
